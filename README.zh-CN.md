## ✨ 核心特性

### 🎯 **零安装，即开即用**
无需下载安装任何软件，直接在浏览器中启动训练。支持所有现代浏览器。

### 🎮 **专业训练模式**
- **目标追踪**：提升你的跟枪精准度
- **甩枪训练**：增强反应速度和爆发力
- **精准模式**：掌握点射精准度
- **速度模式**：训练快速转移目标能力
- **自定义场景**：创建专属训练方案

### ⚙️ **高度自定义**
- 可调节目标大小和速度
- 自定义准星设计
- 个性化难度设置
- 游戏专属灵敏度匹配

### 📊 **性能分析系统**
- 实时精准度追踪
- 进步曲线图表统计
- 个人最佳成绩记录
- 详细训练数据分析

### 🌐 **多语言全球化**
支持中文、英语、日语、葡萄牙语、俄语、韩语、德语、西班牙语等多种语言。

### 🆓 **永久免费**
没有隐藏费用，没有高级会员，没有广告打扰你的训练。

## 🚀 快速开始

使用 AimTrainerX 非常简单：

1. **访问** [aimtrainerx.com/zh-CN](https://aimtrainerx.com/zh-CN)
2. **选择** 你的训练模式
3. **配置** 你的设置（可选）
4. **开始** 立即训练！

### 第一次使用？

1. 从下拉菜单中选择你的游戏
2. 输入你的游戏内灵敏度
3. 选择适合新手的训练模式
4. 点击"开始训练"

**就这么简单！** 你现在正在提升你的瞄准能力。

## ❓ 常见问题

### **AimTrainerX 真的免费吗？**
是的！100% 免费，没有任何隐藏费用或高级功能收费。

### **需要下载安装吗？**
不需要！AimTrainerX 完全在网页浏览器中运行。

### **支持哪些浏览器？**
所有现代浏览器：Chrome、Firefox、Safari、Edge 和 Opera。

### **可以在手机上使用吗？**
目前优化了桌面端体验。移动端支持正在开发中。

### **如何导入游戏灵敏度？**
在设置菜单中选择你的游戏，然后输入灵敏度数值。我们会自动处理转换。

**还有问题？** 查看我们的[完整FAQ](docs/FAQ.md)或[提交问题](https://github.com/LinightKira/aimtrainerx/issues/new)。
